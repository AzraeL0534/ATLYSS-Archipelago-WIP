# ATLYSS Archipelago - Project Roadmap

## Completed Features

### Core Integration
- Full Archipelago multiworld connectivity
- Bidirectional communication with Archipelago servers
- BepInEx plugin implementation
- Real-time item delivery system
- Location check tracking and submission

### Items (137 Total)
- Weapons: Swords, Bows, Katars, Hammers, Spears, Bells, Wands, Guns, Scythes
- Armor: Helms, Chestpieces, Leggings, Capes, Shields, Rings
- Consumables: Health/Mana items, Stat potions, Experience items
- Trade Items: Badges, Clusters, Ingots, Bonds
- Progression Items: Portal unlocks, Class Tomes, Skill Scrolls
- Currency: Crown drops (Small, Medium, Large, Huge)

### Location Checks
- Quest completion tracking (all available quests)
- Level-up detection (levels 1-32)
- Automatic check submission

### Goal Options
- Level-based goals: Reach level 4, 8, 16, 24, or 32
- Boss defeat goals: Colossus, Galius, Lord Kaluuz, Valdur


### Quality of Life
- Item drop queue system with visual feedback
- In-game chat notifications
- F5 quick connect/disconnect
- Automatic progression tracking
- Configuration file support


## Current Work

### Shop Integration
- Shop Sanity option (disabled/enabled)
- Randomized shop inventory when enabled

### Area Access System
- Locked mode: Portal items required to unlock dungeons
- Unlocked mode: All areas accessible from start
- Progressive mode: Sequential portal unlocking (Catacombs then Grove)
- Continuous enforcement system to prevent unintended access


### Testing and Refinement
- Portal locking system verification
- Cross-game multiworld testing
- Performance optimization
- Bug fixes and stability improvements
